    1、安装nodejs

     https://nodejs.org  下载node.js
$ sudo su   #切换到root用户，输入密码
$ node -v   #node版本
$ npm -v    #npm版本

    2、安装hexo

    $ npm install -g cnpm --registry=https://registry.npm.taobao.com                    淘宝源安装cnpm

$ cnpm install -g hexo-cli                                                          安装hexo

$ cnpm -v                                                                           cnpm版本

$ hexo -v                                                                           exo版本

    3、搭建博客

    $ pwd                       #查看当前路径
$ mkdir blog                #创建文件夹
$ cd blog                   #进入文件夹

$ sudo hexo init            #初始化 hexo
$ hexo s                    #启动hexo


$ hexo new "My New Post"    #创建新帖子
$ hexo server               #运行服务
$ hexo generate             #生成静态文件
$ hexo deploy               #部署

    4、修改博客

    $ hexo n "文章题目"         #创建文章
$ vim 文章题目.md           #使用vim打开文件
esc > $wq >                #保存退出
$ cd ../..                 #目录退两层
$ hexo clean               #清理
$ hexo g                   #生成

    6、发布到github

    • Create a new repository

    • Repository name：hollow365x.github.io

      $ cnpm install --save hexo-deployer-git

    type: git

branch: master
    • 部署

    $ hexo clean
    $ hexo g
